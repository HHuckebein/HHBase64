  • Base64

    Provides base-64 en-/decoding for the standard and url safe alphabet. Encoding and Decoding follows RFC4648 which means e.g. that an error is thrown if an encoded strings contains illegal characters. When decoding .urlSafe encoded strings padding characters are ignored.
